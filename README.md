# console-hangman-java
A console-based Hangman game in Java using file I/O and random word selection.


This project is a Java console Hangman game where words are loaded from an external text file. The game randomly selects a word, tracks user guesses, displays hangman stages, and ends with a win or loss based on incorrect attempts. Built to practice core Java concepts such as file handling, ArrayList, loops, and conditional logic.
